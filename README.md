# SQL Data Cleaning & Exploratory Data Analysis

## Objective

Analyze a retail sales dataset using SQL to understand its structure, validate the data, perform cleaning, and generate business insights.

## Tools

- MySQL Workbench
- SQL

## Dataset

- Superstore Sales Dataset
- 9,693 Records
- 21 Columns

## Key Findings
The West region generated the highest sales.

Technology was the highest-performing category by profit.

A small number of customers contributed a large share of total sales.

Higher discounts generally corresponded to lower average profit.
